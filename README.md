# 📊 E-Commerce Data Analysis Project

This project analyzes e-commerce category performance using SQL, Excel, and Tableau.

## 🔧 Tools Used
- SQL (data extraction)
- Excel (data cleaning & preparation)
- Tableau (data visualization)

## 📊 Analysis
- Revenue by product category
- Shipping cost comparison
- Profit margin analysis

## 💡 Key Insights
- Beleza_saude generates the highest revenue
- Cama_mesa_banho has the highest shipping cost
- Relogios_presentes shows the highest profit margin

## 📌 Conclusion
High revenue does not always mean high profitability due to shipping costs.
